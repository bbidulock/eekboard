eekboard by ueno

a virtual keyboard for GNOME

eekboard is a virtual keyboard software package which ships with a standalone virtual keyboard application (eekboard), and a library to create keyboard-like UI (libeek).

Screenshots

Download

  • eekboard-0.0.2.tar.gz

You can also clone the project with Git by running: 

$ git clone git://github.com/ueno/eekboard

Installation

  • Requirements: GLib, GTK+, libxklavier, libfakekey
  • Optional: Clutter, Clutter-Gtk 
$ ./configure
$ make
$ sudo make install
$ eekboard
Some version combinations of Clutter and Clutter-Gtk do not cooperate well. If you get something strange, try "EEKBOARD_DISABLE_CLUTTER=1 eekboard".
